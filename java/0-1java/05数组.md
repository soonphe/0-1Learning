## 数组

### 简介
* 数组是编程语言中常见的一种数据结构，可以存储多个数据，每个数组元素存放一个数据，通常可以通过数组元素的索引来访问数组元素，包括赋值，取值
* Java中的数组要求所有的数组元素具有相同的数据类型。因此在一个数组中，数组元素的类型是唯一的，即一个数组只能存储一种数据类型的数组。
* Java的数组即可以存储基本类型的数据，也可以存储引用类型的数据，只要数组元素的类型统一即可。

    * 数组的长度 -- length

* 定义数组的语法
    * int[] myArray；

### 创建数组步骤
``````
//声明数组    
int[ ] myNumbers;
//构造数组
myNumbers = new int[20];
//初始化数组
myNumbers [ 0 ] = 5;
myNumbers [ 1 ] = 10;
myNumbers [ 2 ] = 15;
myNumbers [ 3 ] = 20;
``````

### 数组的内存模型

int[ ] array = new int [4] ;
* 数组是存储多个相同类型变量的对象。数组的所有元素保存在堆内存中。
* 创建一个数组就是在堆中创建一个数组对象。
* 数组创建后立即拥有默认值。
* 索引从0开始。
* 连续分配

### 数组的初始化
* 静态初始化

    * int[ ] point = new int[ ]{1,2,3,4};
* 动态初始化
    * int[ ] point = new int[3];

### 数组的长度

### 多维数组
* 声明数组
    * boolean[ ][ ] isFull; boolean isFull[ ][ ]; boolean [ ]isFull[ ];

    * boolean[ ][ ]	isFull = new boolean[4][3];
* 初始化

    * int[ ][ ] point = new
    * int[][]{{1,2,3},{4,5,6},{7,8,9}};
* 多维数据的遍历


### 锯齿数组

* 数组行的长度不同，我们称之为锯齿数组；
* 静态初始
    * int[ ][ ] point = {{1,2,3}, {4,5}, {7}};
* 动态初始化
    * int[ ][ ] point = new int[3][ ];
    * point[0] = new int[2]; 
    * point[1] = new int[3]; 
    * point[2] = new int[4];
–	必须指定第1维的长度
