## 数据库问题集锦

#### 报错信息：Too many connections。。。

查看数据库连接数sql：
show status where variable_name = 'threads_connected';

查看数据库连接数详情：
select id,
user,
host,
db,
command,
time,
state,
info
from information_schema.processlist;
